# BookStore (Library Management)

Simple client-side library/bookstore project.

## Files
- index.html - UI
- style.css - styles
- script.js - app behavior
- storage.js - data persistence functions
- db.json - default data

## Run
1. Open index.html in a browser.
2. Or run local server: https://rainbow-cobbler-5c36ff.netlify.app/
