# 📚 PageTurn — Modern Bookstore App

Simple client-side library/bookstore project.

## 🌟 Features
- User Authentication — Sign up and sign in with email & password; secure session management
- Book Library — Powered by the Open Library API with access to millions of titles
- Smart Search — Search books by title, author, or keyword in real time
- Genre Browsing — Filter books by categories like Fiction, Mystery, Sci-Fi, History, Philosophy, Biography, and Horror
- Author Shortcuts — Quick-access marquee with popular classic authors (Shakespeare, Austen, Orwell, etc.)
- Bookmarks — Save favourite books and access them anytime from the Bookmarks tab
- Reading History — Automatically tracks books you've opened
- User Profile — Displays name, email, join date, bookmark count, and books read
- Responsive Design — Clean, modern UI that works across devices
- 📦 All app data is stored locally in the browser using localStorage in JSON format — no backend required

  ## Tech Stack
  - Frontend — HTML, CSS, JavaScript
  - Auth — Custom email/password authentication
  - Book Data — Open Library API
